## 编写源码

先执行`touch hello.cpp`创建文件

再执行`vim hello.cpp`对文件进行编辑（如果你没有vim这条命令可以尝试nano、vi、edit替换）

```
//hello.cpp
#include <cstdio>

int main(){
	    using namespace std;
        printf("hello world\n");
        return 0;
}
```

## 预处理

我们执行`g++ -E hello.cpp -o hello.i`，-o表示输出文件名为`hello.i`

## 编译

使用`g++ -0g -S hello.cpp`进行编译
